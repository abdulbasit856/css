CSS Selectors and Properties

Selectors

  1. 1. Universal Selector (*)
    Selects all elements in the document.
    * { margin: 0; padding: 0; }
  2. 2. Type Selector (element)
    Selects all elements of a given type (e.g., div, p, h1).
    p { color: blue; }
  3. 3. Class Selector (.classname)
    Selects elements with a specific class.
    .highlight { background-color: yellow; }
  4. 4. ID Selector (#idname)
    Selects an element with a specific ID.
    #header { font-size: 24px; }
  5. 5. Attribute Selector ([attribute])
    Selects elements with a specific attribute.
    input[type="text"] { border: 1px solid #ccc; }
  6. 6. Descendant Selector (ancestor descendant)
    Selects elements that are descendants of a specified element.
    div p { margin: 10px; }
  7. 7. Child Selector (parent > child)
    Selects direct children of a specified element.
    ul > li { list-style-type: none; }
  8. 8. Adjacent Sibling Selector (A + B)
    Selects an element that is directly after a specified element.
    h1 + p { margin-top: 0; }
  9. 9. General Sibling Selector (A ~ B)
    Selects all siblings after a specified element.
    h1 ~ p { color: green; }
  10. 10. Pseudo-class Selectors (:pseudo-class)
    Selects elements based on their state.
    a:hover { text-decoration: underline; }
  11. 11. Pseudo-element Selectors (::pseudo-element)
    Selects a part of an element.
    p::first-line { font-weight: bold; }

Properties

Color Properties

  • 1. color: Sets the text color.
    color: red;
  • 2. background-color: Sets the background color.
    background-color: blue;
  • 3. border-color: Sets the border color.
    border-color: green;
  • 4. outline-color: Sets the outline color.
    outline-color: orange;

Text Properties

  • 1. font-family: Specifies the font for text.
    font-family: Arial, sans-serif;
  • 2. font-size: Sets the size of the font.
    font-size: 16px;
  • 3. font-style: Sets the style of the font (normal, italic, etc.).
    font-style: italic;
  • 4. font-weight: Sets the thickness of the font.
    font-weight: bold;
  • 5. line-height: Sets the space between lines of text.
    line-height: 1.5;
  • 6. text-align: Sets the horizontal alignment of text.
    text-align: center;
  • 7. text-decoration: Sets decoration on text (underline, overline, etc.).
    text-decoration: underline;
  • 8. text-transform: Controls text case (uppercase, lowercase, etc.).
    text-transform: uppercase;
  • 9. letter-spacing: Sets space between letters.
    letter-spacing: 0.1em;
  • 10. word-spacing: Sets space between words.
    word-spacing: 0.2em;

Box Model Properties

  • 1. margin: Sets the space outside an element.
    margin: 20px;
  • 2. padding: Sets the space inside an element.
    padding: 10px;
  • 3. border: Sets the border around an element.
    border: 1px solid black;
  • 4. border-radius: Sets rounded corners for an element.
    border-radius: 5px;
  • 5. box-shadow: Adds shadow effects around an element.
    box-shadow: 2px 2px 5px gray;
  • 6. width: Sets the width of an element.
    width: 100px;
  • 7. height: Sets the height of an element.
    height: 50px;
  • 8. max-width: Sets the maximum width of an element.
    max-width: 300px;
  • 9. max-height: Sets the maximum height of an element.
    max-height: 200px;
  • 10. min-width: Sets the minimum width of an element.
    min-width: 50px;
  • 11. min-height: Sets the minimum height of an element.
    min-height: 30px;

Positioning Properties

  • 1. position: Sets the type of positioning for an element.
    position: relative;
  • 2. top: Sets the top position of a positioned element.
    top: 10px;
  • 3. right: Sets the right position of a positioned element.
    right: 20px;
  • 4. bottom: Sets the bottom position of a positioned element.
    bottom: 30px;
  • 5. left: Sets the left position of a positioned element.
    left: 40px;
  • 6. z-index: Sets the stack order of an element.
    z-index: 10;

Display Properties

  • 1. display: Specifies the display behavior of an element.
    display: block;
  • 2. visibility: Specifies whether an element is visible or hidden.
    visibility: hidden;
  • 3. overflow: Specifies what happens when content overflows an element's box.
    overflow: auto;
  • 4. flex: Specifies the flex properties for a flex container.
    flex: 1;
  • 5. grid: Specifies the grid properties for a grid container.
    grid-template-columns: repeat(3, 1fr);
  • 6. float: Specifies how an element should float.
    float: left;
